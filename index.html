Project Abstract

ASTRA (Agri Smart Tech Robotic Automation) is an intelligent pesticide sprinkling prototype that minimizes pesticide wastage and improves crop health. Using cameras and multispectral / NPK sensors, ASTRA runs an edge ML model to detect infected regions and commands a robotic sprayer to apply pesticide precisely. Telemetry syncs to a cloud dashboard and mobile app for monitoring and scheduling.

Goals: precision pest control • reduced chemical footprint • automated monitoring • scalable across crops

Problem Statement

Conventional pesticide spraying is broad and wasteful, harming beneficial organisms and increasing costs. Farmers lack real-time, localized diagnostics; ASTRA empowers targeted intervention by fusing sensor data with on-edge intelligence and robotic actuation.

Solution — Working Principle

Three Intelligent Layers

  1. Sensing: RGB & multispectral cameras, NPK, humidity and ambient sensors collect field data.
  2. Decision: On-device ML generates infection masks and confidence scores.
  3. Actuation: Robotic platform navigates and micro-target sprayers apply controlled doses only where needed.

Data & UX

Telemetry is logged to a cloud dashboard and mobile app — showing infection maps, pesticide usage, and soil health over time for data-driven decisions.

Key Features

Precision Spraying

Mask-driven nozzle control reduces chemical use and prevents over-spray.

Edge AI Detection

Efficient CNN models for in-field inference on Raspberry Pi / Jetson devices.

IoT Telemetry

Environmental and NPK sensors feed analytics and alerts.

Mobile App

Live monitoring, run scheduling and historical reports.

Eco-Friendly

Reduces runoff and protects beneficial species.

Scalable

Modular hardware supports drones, tractors and small robots.

Hardware & Software

Hardware

Raspberry Pi / Jetson Nano (edge)
RGB + multispectral camera
NPK sensor
Ultrasonic / LiDAR
Precision solenoid nozzles & pump
Chassis + power management

Software

  • Edge ML: TensorFlow Lite / PyTorch Mobile
  • Control: ROS / microcontroller drivers
  • Backend: MQTT + REST, cloud DB & analytics
  • Mobile: React Native / Flutter dashboard

System Architecture

Diagram notes: Camera & sensors → edge inference → decision mask → sprayer actuator. Telemetry → cloud dashboard & mobile app.

Future Scope

  • Multi-crop adaptation with transfer learning
  • Drone & tractor integration for large farms
  • Weather-aware scheduling via forecasts
  • Federated learning for model privacy & improvement
  • Automated refill & docking for continuous operation
